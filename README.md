# DSA Practice Project

A small TypeScript project for practicing core data structures and algorithms from the Frontend Masters course.

## What's in this project

- Search algorithms (Linear Search, Binary Search)
- Sorting (Bubble Sort)
- Math/problem-solving examples (Binary Square Root, Two Crystal Balls)
- Data structures (Linked List, Linked Node)

All source files are in `src/`.

## Run locally

1. Install dependencies:
   `npm install`
2. Build TypeScript:
   `npm run build`
3. Run compiled output (if your scripts support it):
   `npm start`

## Goal

Keep each algorithm simple, readable, and easy to test while learning DSA fundamentals.
